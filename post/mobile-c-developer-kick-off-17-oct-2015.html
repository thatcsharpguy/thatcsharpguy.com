Mobile C# developer kick-off (17-oct-2015)

Este es un post para recapitular lo visto en el evento llevado a cabo en la Escuela Superior de Cómputo.

En este repositorio de git está el código de la aplicación desarrollada durante el taller aquí está la explicación del algoritmo de Levenshtein y la implementación usada en C#.

Esta es el post en el blog de Miguel de Icaza en donde habla sobre Mono en el PS4.

De los plugins y extensiones:

  • Akavache
  • Polly
  • MVVM light
  • Plugin Connectivity
  • Plugin TextToSpeech
  • XLabs

Para la gran mayoría que se quedó sin libro, no olviden el reto de la Sharpuladora y si aún así no se resisten las ganas de tenerlo, acá está la versión electrónica.
